# Quick Start

Get Delimiter monitoring your AI rate limits in under 2 minutes.

## 1. Sign in at delimiter.app

GitHub OAuth — one click, you're in. No passwords, no separate sign-up step.

## 2. Copy your project key

Your project key (starts with `dlm_`) is shown on the dashboard after sign in.

This key identifies your project — it's not a secret, but keep it private.

## 3. Install the SDK

```bash
npm install @delimiter/sdk
```

## 4. Initialize Delimiter

```typescript
import { delimiter } from '@delimiter/sdk'
import OpenAI from 'openai'

delimiter.init('dlm_your_project_key')

// Use your AI clients as normal — Delimiter monitors automatically
const openai = new OpenAI({ apiKey: process.env.OPENAI_KEY })

const response = await openai.chat.completions.create({
  model: 'gpt-4o',
  messages: [{ role: 'user', content: 'Hello' }]
})
```

That's it. Two lines added to your existing code — `import` and `init`. Delimiter hooks into the HTTP layer and monitors all AI API calls automatically.

## 5. Dashboard lights up

The moment your first API call completes, your dashboard shows real-time rate limit data. Health cards appear for each provider, showing your current usage against the provider's limits.

## 6. (Optional) Connect providers for credit monitoring

For spend monitoring, connect your provider accounts in the dashboard under **Connections**. Delimiter never stores your credentials — you authenticate directly with each provider, and Delimiter receives read-only billing data.
