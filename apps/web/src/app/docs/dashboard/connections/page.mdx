# Provider Connections

Connect your AI provider accounts to monitor spend and prevent blackouts from depleted funds.

## How it works

When you connect a provider:

1. You authenticate directly with the provider in a secure popup
2. Delimiter never sees or stores your credentials
3. Authenticated billing API calls are proxied securely on your behalf
4. Delimiter receives only billing data (balances, spend) — never API keys

## Supported providers

| Provider | Key Type | Capabilities |
|----------|----------|-------------|
| **OpenAI** | Admin API Key | Period spend |
| **Anthropic** | Admin API Key | Period spend |
| **OpenRouter** | API Key | Balance, period spend |
| **xAI (Grok)** | Management Key (Read-only) | Balance |

More providers are added regularly.

## Connecting a provider

1. Go to **Dashboard → Connections**
2. Click **Connect** next to the provider
3. Authenticate with the provider in the popup
4. Click **Refresh** to fetch initial billing data

Once connected, billing data is polled automatically every 15–30 minutes.

## Security

- **Your keys stay with you.** You authenticate directly with each provider. Delimiter never receives your API keys or credentials.
- **Read-only access.** Delimiter only requests billing/usage data — it cannot create API keys, make API calls on your behalf, or modify your account.
- **Disconnect anytime.** Click Disconnect to immediately revoke access.
